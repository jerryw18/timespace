# TimeSpace Frontend

A historical timeline and mapping application built with React, TypeScript, and Vite. Features an interactive map showing historical events with a timeline interface.

## Tech Stack

- **React 19** with TypeScript
- **Vite** for build tooling and development
- **Tailwind CSS** for styling
- **Axios** for HTTP requests

## Development

### Prerequisites

- Node.js 18+
- npm

### Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## Docker

### Build and Run with Docker

```bash
# Build the Docker image
docker build -t timespace-frontend .

# Run the container
docker run -p 3000:80 timespace-frontend
```

The application will be available at http://localhost:3000

### Docker Details

- Uses multi-stage build for optimized production image
- Build stage: Node.js 18 Alpine for dependency installation and building
- Production stage: nginx Alpine for serving static files
- Final image size is minimal (~25MB)
